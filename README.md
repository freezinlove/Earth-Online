<h1 align="center">献给热爱旅行的你</h1>

<p align="center">
  一座由 AI 驱动的私人旅行星球。
</p>

<p align="center">
  Earth Online 自动会理解你的照片、辨认旅途中的地点、整理时间与路线，并把散落在硬盘里的回忆重新点亮在 3D 地球和时间线上
</p>

<br />

<p align="center">
  <strong>准备环境</strong>
</p>

<p align="center">
  需要 Node.js 24+ 和 npm。Windows 可用下面的命令安装：
</p>

```powershell
winget install OpenJS.NodeJS
```

```bash
node --version
npm --version
```

<p align="center">
  <strong>启动项目</strong>
</p>

```bash
npm ci
npm run dev
```

<p align="center">
  打开 <code>http://localhost:5173/</code>
</p>

<p align="center">
  <strong>使用建议</strong>
</p>

<p align="center">
  尽量导入带有GPS信息的照片
</p>
