# Earth_Online 抽象粒子地球方案

## 设计目标

做一个“记忆星图式地球”，不是传统卫星地球：

- 抽象、艺术化、粒子点阵。
- 不依赖真实地球贴图作为主视觉。
- 放大不模糊，因为主要由几何点、线、shader 构成。
- 地理准确：远景大洲，中景国家，近景省级边界。
- 城市边界暂时不做，城市用点和标签表达。
- 旅行地点、照片点、路线始终是视觉主角。
- 细节随缩放平滑出现和消失，接近无级调节。

## 地球视觉结构

### 基础球体

使用一个深色、半透明、低反射的球壳提供体积感。它不显示真实地表纹理，或者只使用极弱的暗色纹理作为辅助。

### 陆地点阵

用陆地轮廓数据生成粒子点阵，粒子构成大陆和岛屿形状。

粒子颜色不局限于暖金色：

- 冷青、蓝紫、灰绿作为基础陆地点。
- 当前旅行区域用珊瑚、玫红、琥珀、电蓝高亮。
- 历史旅行区域低透明保留痕迹。

### 边界层

边界不是一直全部显示，而是随缩放平滑变化：

- 远景：大洲或海岸轮廓。
- 中景：国家边界。
- 近景：省级边界，例如中国省份、日本都道府县、美国州。
- 城市边界：暂时不做。

### 城市与地点层

城市不做边界，用点表达：

- 远景不显示城市，避免杂乱。
- 中近景显示主要城市。
- 近景显示当前旅行相关城市、小镇、照片地点。
- 选中地点显示标签，其他标签隐藏或低透明。

### 旅行记忆层

这是最高优先级：

- 当前旅行地点点位更亮。
- 照片点可以比城市点更小但更有光感。
- 当前路线用流动光点表现。
- 历史路线低透明，像记忆痕迹。
- 点击地点后显示标签、照片数量、时间信息。

## 缩放层级

采用“数据分级，视觉无级”。

数据是分层的，但视觉上通过透明度、粒子大小、发光强度连续变化。

```ts
continentOpacity = 1 - smoothstep(0.25, 0.45, zoom);
countryOpacity =
  smoothstep(0.25, 0.55, zoom) *
  (1 - smoothstep(0.82, 0.96, zoom));
provinceOpacity = smoothstep(0.58, 0.88, zoom);

cityOpacity = smoothstep(0.62, 0.92, zoom);
memoryOpacity = smoothstep(0.08, 0.35, zoom);
```

用户感受到的是：

- 远看：抽象大洲粒子地球。
- 放大：国家边界慢慢浮现。
- 再放大：省级边界慢慢浮现。
- 更近：城市点、旅行点、照片点和路线变清晰。

## 粒子细节

粒子不只是简单变大，而是随缩放逐渐增加密度：

- 预生成不同密度的陆地点。
- 每个粒子有 `revealAt`。
- zoom 越近，越多粒子淡入。
- 粒子 scale、opacity、emissive 强度平滑变化。

这样放大后不是图片变大，而是地理细节变多。

## 路线表现

路线根据距离自动选择形态：

- 短距离：贴球面路径，精准、细腻。
- 中距离：轻微抬高，不夸张。
- 长距离：优雅低弧线。
- 当前路线：带流动粒子或渐变光。
- 历史路线：降低透明度。

路线始终比普通边界更亮，但不要比选中地点抢眼。

## 数据策略

推荐数据源：

- 陆地和海岸线：Natural Earth。
- 国家边界：Natural Earth 或 geoBoundaries ADM0。
- 省级边界：geoBoundaries ADM1，按当前旅行国家或当前视野区域加载。
- 城市点：GeoNames、OpenStreetMap 或本地缓存。
- 旅行点：项目内的 `placeNodes`。
- 照片点：照片 GPS。

关键策略：

- 全球只加载轻量数据：陆地、大洲或海岸、国家。
- 省级边界按需加载，不全球一次性加载。
- 城市点做重要性过滤。
- 照片和旅行地点永远优先显示。
- 地名解析结果要缓存，不依赖实时请求。

## 性能策略

- 粒子用 `BufferGeometry` 或 `InstancedMesh`。
- 不用成千上万个 React 组件表示粒子。
- 缩放时只更新 shader uniform，不每帧重建数据。
- 省级边界懒加载并缓存。
- 边界线做简化，避免过多顶点。
- 当前旅行区域可以更精细，全球区域保持艺术化。

## 推荐最终风格

整体可以是：

> 深色冷调半透明球体 + 多色陆地粒子 + 青蓝/蓝紫国家线 + 近景省级细线 + 珊瑚/电蓝旅行点 + 渐变流光路线

页面背景仍然保持温暖和安静，但地球可以用更强的互补色，让它成为视觉核心。

## 实施顺序

1. 第一版：抽象球体、陆地点阵、国家边界、旅行点、路线、点击标签。
2. 第二版：缩放控制细节，大洲、国家、省级边界平滑显隐，粒子密度随缩放增加。
3. 第三版：当前旅行国家加载省级边界，加入主要城市点和城市标签。
4. 第四版：路线流光、粒子呼吸、当前区域扩散、多色主题 preset。
